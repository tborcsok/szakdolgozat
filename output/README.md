## Regressziós outputok

Itt találhatóak a regressziós outputok az iskolaminőség-indexek létrehozására futtatott modellek és a végső elemzés moddeljei alapján kimentve. A regressziós outputokban a sum10_*x*of5 változónevek alatt többféle index szerepel (E1, E2, I1, I2, F1, F2). A közös névre hozás az összefoglaló táblázat értelmes terjedelmi keretek között tartására szolgál. Az outputok így érdemileg csak a kód böngészése után értelmezhetőek.

A fájlok a STATA 13 statisztikai  programcsomaggal szépen megjeleníthetőek.
